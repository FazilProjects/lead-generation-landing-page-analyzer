## Generated Report Sections

The generated report should include:

- Executive Audit Summary
- Overall Conversion Score + Risk Level
- Top 3 Priority Fixes
- Score Breakdown
- Hero Section Fix
- Offer Clarity Audit
- CTA Strength Checker
- Trust Proof Recommendations
- Lead Form Friction Review
- Mobile Readiness Checklist
- Conversion Risk Analysis
- Suggested Improvements
- Revised Landing Page Structure
- 7-Day Landing Page Improvement Plan
- Exportable Audit Report

## Wording Guidelines

- Keep recommendations practical and direct.
- Mention the offer, audience, CTA, and traffic source dynamically.
- Use client-ready language without claiming real AI analysis.
- Avoid saying the dashboard fetched, crawled, tracked, predicted, or analyzed a live page.
- Use terms like local JavaScript, rule-based scoring, planning dashboard, and portfolio project.
- Keep risk language useful rather than alarmist.
