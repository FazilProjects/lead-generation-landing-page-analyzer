# Project Brief

## Goal

Lead Generation Landing Page Analyzer is a static frontend dashboard that helps users turn a short landing page brief into a practical lead generation landing page audit report.

The first version should feel polished, useful, and portfolio-ready. A user should be able to enter campaign context, click one button, and receive a structured client-style audit.

## Scope

The project includes:

- A responsive dashboard interface
- Landing page brief input
- Business type selection
- Campaign traffic source selection
- Offer, target audience, CTA, and lead form field inputs
- Optional landing page URL text field
- Local rule-based scoring and recommendations
- Exportable text audit report
- Project documentation

The project does not include:

- External APIs
- Live page fetching
- Google Ads or Meta Ads integrations
- Analytics integrations
- Tracking scripts
- Real AI analysis
- Backend storage

## User Workflow

1. User enters a landing page brief.
2. User selects business type.
3. User selects campaign traffic source.
4. User enters offer, target audience, main CTA, and lead form fields.
5. User optionally enters a landing page URL as text only.
6. User clicks Generate Landing Page Audit.
7. Dashboard generates a complete local audit report.
8. User exports the audit report or resets the dashboard.

## Core Features

- Audit summary
- Overall conversion score
- Risk level
- Score breakdown cards
- Hero section analyzer
- Offer clarity audit
- CTA strength checker
- Trust proof checklist
- Lead form friction score
- Mobile readiness checklist
- Conversion risk analysis
- Suggested improvements
- Revised landing page structure
- 7-day landing page improvement plan
- Exportable report with portfolio disclaimer

## Scoring Logic

The scoring logic is intentionally simple and readable:

- Vague or short offers reduce offer clarity.
- Vague CTAs like Submit or Learn More reduce CTA strength.
- Too many form fields reduce the form friction score.
- Trust-related words in the brief improve trust proof.
- Google Ads emphasizes search intent and message match.
- Meta Ads emphasizes hook, offer clarity, proof, and mobile experience.
- Organic traffic emphasizes SEO clarity, trust, and content flow.
- Mixed traffic emphasizes message consistency and segmented CTA paths.

## Limitations

This is a local JavaScript planning dashboard. It should always be presented as a portfolio project, not as a production AI platform or real CRO software product.

It does not validate real landing page performance, inspect live pages, predict actual conversion rates, or replace real analytics.
